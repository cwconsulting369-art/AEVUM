# Willkommen im AEVUM Document-Exchange

Hier kannst du Dokumente mit deinem AEVUM-Agent austauschen.

- **inbox/** — Du lädst Files hier hoch für den Agent
- **outbox/** — Der Agent schreibt Files für dich hier
- **shared/** — Bidirektional, beide können editieren

Markdown wird im Dashboard schön formatiert dargestellt.
