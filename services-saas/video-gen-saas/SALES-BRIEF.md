# SALES-BRIEF — Video-Generation-SaaS

Self-serve, no-call, Stripe-Checkout. Kein Sales-Call nötig (AEVUM-Regel: Cash = self-serve/async).

## Pricing-Tiers

Credit-basiert (1 Credit ≈ 1 Render-Sekunde fertiges Video, vereinfacht). Provider-Realkosten liegen bei ~$0,10–0,75/sec Video + ~$0,11–0,25/Render-Minute + TTS-Credits — Tiers sind so kalkuliert, dass selbst Premium-Modelle Marge lassen.

| Tier | Preis | Credits / Monat | Zielgruppe | Limits / Features |
|---|---|---|---|---|
| **Starter** | **€27/mo** | ~10 Videos (à 30s) | Solo-Creator, Test | 1 Brand-Kit, Kling-B-Roll, Voiceover, Watermark „made with", 9:16/16:9 |
| **Creator** | **€67/mo** | ~30 Videos | Personal Brand, kleine Marke | 3 Brand-Kits, Avatar-Option, kein Watermark, alle Formate, Caption-Styles |
| **Pro** | **€97/mo** | ~60 Videos | KMU-Marketing | 5 Brand-Kits, Premium-Modelle (Veo/Runway), Priority-Render, Share-Analytics |
| **Agency** | **€297/mo** | ~250 Videos | Agenturen, Volumen | Multi-Brand-Workspaces, Bulk/CSV (Phase 2), API-Zugang, Sub-Accounts |
| **Top-up** | **€19 / Paket** | +50 Credits | alle | überzieht ohne Plan-Wechsel, Margenschutz bei Spikes |

**Jahresabo:** −20% (2 Monate gratis) → Cash vorab, Churn-Schutz.
**Free-Trial:** 3 Test-Videos mit Watermark, keine Kreditkarte → friktionsfreier Self-Serve-Funnel-Einstieg.

> Low-Ticket-Anker €27 (Solo) → klassische Wert-Leiter bis €297 (Agentur). Bewusst **unter** HeyGen/Synthesia-Enterprise, **über** Roh-Tool-Bastellösung.

## Mehrwert / ROI

**Vergleichsanker (was Kunde sonst zahlt):**
- Freelance-Video-Editor: 150–500 € pro fertigem Clip.
- Agentur-Video-Paket: 1.500–5.000 € / Monat.
- AI-Einzeltools gestapelt (HeyGen $24 + ElevenLabs $99 + Render-Tool $41 + manuelles Zusammenschneiden) = >$160/mo **plus** Eigenarbeit/Stundenlohn fürs Orchestrieren.

**Mit diesem SaaS:**
- Pro-Tier €97/mo = ~60 fertige Videos → **~1,60 € pro Marketing-Video**, voll gebrandet, in Minuten.
- Spart die manuelle Orchestrierung dreier Tools (der eigentliche Zeitfresser).

**ROI-Story (KMU, Pro):** Ein einziger zusätzlich gewonnener Kunde durch Social-Video-Ads deckt das Jahresabo. Bei 60 Videos/Monat ersetzt das Tool faktisch einen Teilzeit-Content-Job (>1.000 €/Monat) für €97.

## Conversion-Story (Funnel)

1. **Hook (Landing):** „Vom Script zum fertigen Marken-Video — in Minuten, nicht Tagen." Demo-Video direkt auf der Page (mit dem Tool selbst erstellt = Dogfood-Proof).
2. **Aha (Free-Trial):** 3 Test-Videos ohne Karte. User gibt Stichworte ein → bekommt fertiges, gebrandetes Video → spürt den Wow-Moment selbst.
3. **Friktion-Entferner:** Watermark + Format-Limit im Trial → natürlicher Upgrade-Trigger.
4. **Conversion:** Stripe-Checkout direkt aus dem Render-Screen („Ohne Watermark herunterladen → Starter €27"). No-Call, sofort.
5. **Expansion:** Credit-Verbrauch sichtbar → Top-ups + Tier-Upgrade, wenn Kunde skaliert. Agentur-Tier als Volumen-Sprung.
6. **Retention:** Brand-Kits + Render-History = Switching-Cost. Template-Marketplace (Phase 2) als wiederkehrender Grund zurückzukommen.

**North-Star-Metrik:** „Videos rendered / Workspace / Monat" — direkt korreliert mit Credit-Umsatz und Retention.

**Margenschutz:** Credit-System + model-agnostische Provider-Wahl (günstige Modelle für Lower-Tier) + Hard-Caps verhindern, dass ein Power-User die Marge frisst.
