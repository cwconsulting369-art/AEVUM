# lennoxOS Sales Website

Professional sales website for lennoxOS - AI Business Infrastructure.

## Stack
- React 19 + TypeScript + Vite
- Tailwind CSS v3
- Framer Motion + GSAP
- React Router (HashRouter)

## Pages
- Home (10 service cards, trust bar, process, social proof)
- 10 Service Detail Pages
- About / Process
- Contact

## Deployment
```bash
npm install
npm run build
```

Static output in `dist/` — ready for Vercel, Netlify, or any static host.
